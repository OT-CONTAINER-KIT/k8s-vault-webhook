# Annotations

This section of document contains the information about Kubernetes annotation which will be used for Webhook mutation. Based on these annotations, the secrets will be mutated.

The annotations which are currently supported:-

  • Hashicorp Vault (opens new window)

There are some other annotations which are planned to be implemented in future.

  • AWS Secret Manager (opens new window)
  • Azure Key Vault (opens new window)
  • GCP Secret Manager (opens new window)
